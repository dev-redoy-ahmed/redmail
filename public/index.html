Email Management

POST /api/temp-email/generate
Generate a random temporary email address
Request:
curl -X POST "http://localhost:3000/api/temp-email/generate" \
  -H "X-API-Key: redmail-api-key-2024-production" \
  -H "Content-Type: application/json" \
  -d '{"type": "random"}'
Response:
{
  "success": true,
  "email": "abc123@oplex.online",
  "emailId": "uuid-here",
  "expiresAt": "2024-01-01T12:00:00Z",
  "domain": "oplex.online"
}
POST /api/temp-email/create-custom
Create a custom temporary email address
Request:
curl -X POST "http://localhost:3000/api/temp-email/create-custom" \
  -H "X-API-Key: redmail-api-key-2024-production" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "myemail",
    "domain": "oplex.online",
    "type": "custom"
  }'
Response:
{
  "success": true,
  "email": "myemail@oplex.online",
  "emailId": "uuid-here",
  "expiresAt": "2024-01-01T12:00:00Z",
  "domain": "oplex.online"
}

Message Management

GET /api/inbox/:emailId
Get inbox messages for a specific email address
Query Parameters:
  • page - Page number (default: 1)
  • limit - Messages per page (default: 50)
Request:
curl -X GET "http://localhost:3000/api/inbox/uuid-here?page=1&limit=10" \
  -H "X-API-Key: redmail-readonly-key-2024"
Response:
{
  "success": true,
  "messages": [
    {
      "id": "msg-uuid",
      "from": "sender@example.com",
      "subject": "Test Email",
      "receivedAt": "2024-01-01T10:30:00Z",
      "isRead": false,
      "hasAttachments": true
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total": 25,
    "pages": 3
  },
  "unreadCount": 5
}
GET /api/message/:messageId
Get detailed information about a specific message
Request:
curl -X GET "http://localhost:3000/api/message/msg-uuid" \
  -H "X-API-Key: redmail-readonly-key-2024"
Response:
{
  "success": true,
  "message": {
    "id": "msg-uuid",
    "from": "sender@example.com",
    "to": "myemail@oplex.online",
    "subject": "Test Email",
    "content": "Email content here...",
    "receivedAt": "2024-01-01T10:30:00Z",
    "isRead": true,
    "attachments": [
      {
        "filename": "document.pdf",
        "size": 1024,
        "downloadUrl": "/api/attachment/att-uuid"
      }
    ]
  }
}
DELETE /api/message/:messageId
Delete a specific message
Request:
curl -X DELETE "http://localhost:3000/api/message/msg-uuid" \
  -H "X-API-Key: redmail-api-key-2024-production"
Response:
{
  "success": true,
  "message": "Message deleted successfully"
}
DELETE /api/inbox/:emailId/clear
Clear all messages from an email inbox
Request:
curl -X DELETE "http://localhost:3000/api/inbox/uuid-here/clear" \
  -H "X-API-Key: redmail-api-key-2024-production"
Response:
{
  "success": true,
  "deletedCount": 15,
  "message": "All messages cleared successfully"
}

Search & Statistics

GET /api/search/messages
Search messages across all emails
Query Parameters:
  • q - Search term (required)
  • page - Page number (default: 1)
  • limit - Results per page (default: 20)
Request:
curl -X GET "http://localhost:3000/api/search/messages?q=test&page=1&limit=5" \
  -H "X-API-Key: redmail-readonly-key-2024"
Response:
{
  "success": true,
  "results": [
    {
      "id": "msg-uuid",
      "from": "sender@example.com",
      "subject": "Test Email",
      "snippet": "This is a test message...",

    }
  ],
  "pagination": {
    "page": 1,
    "limit": 5,
    "total": 12,
    "pages": 3
  }
}
